# How to Enable Shaders?

Enabling shaders requires a compatible rendering mod/plugin. There are two mainstream solutions:

## Option 1: Sodium + Iris Shaders

For Fabric/Quilt loaders:

- **Sodium** (or Rubidium / Embeddium) — Performance optimization rendering mod
- **Iris Shaders** (or Oculus) — Shader loading mod

1. Install Sodium and Iris Shaders mods
2. Launch the game → **Options → Shader Settings**
3. Select an imported shader pack
4. Adjust shader parameters and start playing

## Option 2: Optifine

For Forge loader or standalone installation:

1. Install Optifine (HD Fix)
2. Launch the game → **Options → Video Settings → Shaders**
3. Select a shader pack and apply

:::warning Compatibility
- Optifine has poor mod compatibility in higher versions; not recommended for use with many mods
- Recommended to use Optifine for 1.16.5 and below; Sodium + Iris recommended for 1.17+
- For shader compatibility, refer to the [Shader Compatibility Matrix](/docs/mobileglues/ShaderSupportMatrix)
:::
