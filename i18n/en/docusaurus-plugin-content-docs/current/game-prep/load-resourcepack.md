# How to Load Resource Packs?

Before launching the game, make sure you have properly loaded your resource packs:

1. Select the corresponding version in your launcher and go to version settings
2. Find **Resource Pack Management**
3. Make sure the target resource pack is enabled and in the correct priority position
4. Launch the game and confirm the loading status in **Options → Resource Packs**

:::info Tip
The loading order of resource packs affects the final result. Packs placed higher take priority.
:::
