# How to Install Your Launcher?

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. Go to [FCL Download Site](https://foldcraftlauncher.cn/down.html) or [GitHub Releases](https://github.com/FCL-Team/FoldCraftLauncher/releases) to download the latest APK
2. Install the APK on your phone (you may need to enable "Unknown sources" installation)
3. Open FCL and grant the necessary permissions

:::tip Recommended Channel
[Ningmo Mirror](https://miawa.cn) also provides downloads for FCL, ZL, ZL2, renderers, and JAVA runtimes.
:::

</TabItem>
<TabItem value="zl2" label="ZL2">

1. Go to [ZL Official Site](https://www.zalithlauncher.cn/download) to download the ZL2 installer
2. You can also download from [GitHub](https://github.com/ZalithLauncher/ZalithLauncher2/releases)
3. Install and open, grant necessary permissions

</TabItem>
<TabItem value="zl" label="ZL">

1. Go to [ZL Official Site](https://www.zalithlauncher.cn/download) to download the ZL installer
2. You can also download from [GitHub](https://github.com/ZalithLauncher/ZalithLauncher/releases)
3. Install and open, grant necessary permissions

</TabItem>
</Tabs>
