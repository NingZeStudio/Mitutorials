# How to Configure Your Launcher?

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

After installation, you need to complete basic configuration before you can play normally.

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. Open FCL and go to **Settings**
2. Configure the following key items:
   - **JAVA Version**: Select an installed JAVA runtime
   - **Memory Allocation**: It's recommended to disable auto-allocation; around 1G for vanilla
   - **Version Isolation**: It's recommended to enable
   - **Renderer**: Choose based on your needs (kw for vanilla, MG for mods)

For detailed configuration, refer to [FCL Quick Start](/docs/fcl/intro#游戏配置)

</TabItem>
<TabItem value="zl2" label="ZL2">

1. Open ZL2 and go to the settings page
2. Configure JAVA path and memory allocation
3. Select a renderer scheme
4. Enable version isolation

</TabItem>
<TabItem value="zl" label="ZL">

1. Open ZL and go to settings
2. Set JAVA path
3. Adjust memory allocation
4. Choose an appropriate renderer

</TabItem>
</Tabs>

:::warning Memory Reminder
"More memory = more instability!" — Allocate moderately.
:::
