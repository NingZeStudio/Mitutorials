# Guide / Must Read

Welcome to the LogShare & Minecraft: Java Edition for Android documentation!

Before you begin, please read the following carefully to help you better understand and use the information provided on this site.

:::tip 💡 Core Tips
Playing Minecraft JAVA Edition on mobile relies on the cooperation of a **launcher** and a **renderer**, which is more challenging to get started with than the NetEase version. If you have patience, the mobile JAVA Edition community welcomes you.
:::

:::warning ⚠️ Performance Notice
Playing Minecraft JAVA Edition on mobile is like any other mobile game — don't expect a phone with performance equivalent to a last-century computer to run smoothly.
:::

## Document Navigation

This section will guide you through the following steps:

1. **Choose a Launcher** — Understand the differences between FCL, ZL2, and ZL and choose the one that suits you
2. **Install & Configure** — Download, install, and complete the basic configuration of your launcher
3. **Renderer Setup** — Install and configure MG renderer and other rendering solutions
4. **Resource Management** — Download and import mods, resource packs, shaders, and modpacks
5. **Game Preparation** — Final checks and settings before launching the game

## Important Reminders

- **Mod Compatibility**: Not all mods can run on mobile; some have platform restrictions or special environment requirements
- **Memory Allocation**: It's recommended to disable auto-allocation; vanilla typically needs around 1G. Remember: "More memory = more instability"
- **Cooling**: Use a heatsink or limit the frame rate cap to avoid overheating
