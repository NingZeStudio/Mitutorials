# Choose Your Launcher

There are three mainstream launchers for running Minecraft: Java Edition on Android:

| Launcher | Full Name | Features |
|----------|-----------|----------|
| **FCL** | Fold Craft Launcher | Active community, frequent updates, rich features |
| **ZL2** | Zalith Launcher 2 | Modern UI, easy to use |
| **ZL** | Zalith Launcher | Classic & stable, lightweight & compatible |

## Launcher Overview

### FCL (Fold Craft Launcher)

- Most active community with abundant tutorial resources
- Supports multiple renderer switching
- Built-in mod/resource pack/shader downloads

### ZL2 (Zalith Launcher 2)

- Brand new UI design with smooth experience
- More beginner-friendly
- Continuously being updated

### ZL (Zalith Launcher)

- Classic version with good stability
- Lower resource usage
- Suitable for lower-end devices
