# 外部下载你所需要的Minecraft相关资源（仅给出浏览器下载渠道）

如果你在启动器内找不到需要的资源，可以通过以下浏览器渠道下载后手动导入。

## 如何下载模组文件？

### Modrinth

[Modrinth](https://modrinth.com/mods) — 开源友好的模组平台，搜索和下载体验良好。

### CurseForge

[CurseForge](https://www.curseforge.com/minecraft/mc-mods) — 最大的模组托管平台，资源最全。

### 其他渠道

- **爱发电** — 部分国内作者在此发布模组，需自行查找

## 如何下载资源包？

- [Modrinth - 资源包](https://modrinth.com/resourcepacks)
- [CurseForge - 资源包](https://www.curseforge.com/minecraft/texture-packs)
- 爱发电等（自行查找）

## 如何下载光影文件？

- [Modrinth - 光影](https://modrinth.com/shaders)
- [CurseForge - 光影](https://www.curseforge.com/minecraft/shaders)
- 爱发电等（自行查找）

## 如何下载整合包？

- [Modrinth - 整合包](https://modrinth.com/modpacks)
- [CurseForge - 整合包](https://www.curseforge.com/minecraft/modpacks)
- 爱发电等（自行查找）

:::tip 下载后
从外部下载的资源文件，请参考对应的导入教程：
- [导入模组](/docs/getting-started/import-mods)
- [导入资源包](/docs/getting-started/import-resourcepack)
- [导入光影](/docs/getting-started/import-shader)
- [导入整合包](/docs/getting-started/import-modpack)
:::
