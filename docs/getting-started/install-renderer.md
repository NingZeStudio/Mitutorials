# 如何安装渲染器插件？

渲染器是手机运行 Minecraft: Java Edition 的核心组件，它负责将游戏的 OpenGL 指令翻译为手机 GPU 可执行的指令。

## 渲染器类型

| 渲染器 | 适用场景 | 特点 |
|--------|----------|------|
| **MG (Mobile Glues)** | 模组/类原版 | 兼容性好，社区支持活跃 |
| **Zink (Vulkan)** | 光影支持 | 支持完整光影，性能取决于 Vulkan 驱动 |
| **kw** | 原版游戏 | FCL 默认渲染器，轻量高效 |

## 安装步骤

1. 前往 [下载页面](/docs/download) 获取渲染器插件 APK
2. 安装插件（部分渲染器需要作为启动器插件安装）
3. 在启动器设置中选择已安装的渲染器

详细配置请参考 [MobileGL 文档](/docs/mobileglues)
