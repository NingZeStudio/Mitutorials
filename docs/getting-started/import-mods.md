# 如何导入对应版本的模组文件？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

如果你已经从外部渠道下载了模组文件（.jar），可以手动导入。

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL → **游戏管理** → 选择对应版本
2. 进入**模组管理**
3. 点击**导入**，选择下载的 .jar 文件
4. 确保模组版本与游戏版本匹配

:::warning 版本匹配
模组必须与 Minecraft 版本及加载器版本兼容，否则可能导致崩溃。
:::

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 将 .jar 文件放入对应版本的 mods 目录
2. 或在 ZL2 模组管理中导入

</TabItem>
<TabItem value="zl" label="ZL">

1. 将模组文件复制到版本 mods 目录
2. 在 ZL 中刷新模组列表

</TabItem>
</Tabs>
