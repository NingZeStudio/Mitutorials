# 如何下载对应版本的模组文件？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL → **下载** → **模组**
2. 选择已安装的游戏版本进行筛选
3. 浏览或搜索需要的模组
4. 点击下载，模组将自动保存到对应版本的 mods 目录

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 在 ZL2 中进入模组下载页面
2. 筛选对应版本
3. 下载安装

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL 模组管理
2. 搜索并下载对应版本的模组

</TabItem>
</Tabs>

:::info 外部下载
你也可以从 [Modrinth](/docs/external-download#如何下载模组文件)、[CurseForge](/docs/external-download#如何下载模组文件) 等平台下载后手动导入。
:::
