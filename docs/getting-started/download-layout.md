# 如何下载你所喜欢的键位布局（如果有）？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

键位布局决定了你在手机屏幕上的操作方式。部分启动器支持在线下载社区分享的键位布局。

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL 主界面
2. 进入**控制布局**管理页面
3. 浏览或搜索社区分享的布局
4. 点击下载应用

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 在 ZL2 中进入控制设置
2. 浏览可用布局
3. 下载并应用

</TabItem>
<TabItem value="zl" label="ZL">

1. 进入 ZL 控制设置
2. 查找在线布局资源
3. 下载使用

</TabItem>
</Tabs>
