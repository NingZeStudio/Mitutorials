# 选择你的启动器

目前 Android 平台上运行 Minecraft: Java Edition 的主流启动器有三款：

| 启动器 | 全称 | 特点 |
|--------|------|------|
| **FCL** | Fold Craft Launcher | 社区活跃、更新频繁、功能丰富 |
| **ZL2** | Zalith Launcher 2 | 界面现代化、操作简便 |
| **ZL** | Zalith Launcher | 经典稳定、轻量兼容性好 |

## 各启动器简介

### FCL (Fold Craft Launcher)

- 社区最为活跃，教程资源丰富
- 支持多种渲染器切换
- 内置模组/资源包/光影下载

### ZL2 (Zalith Launcher 2)

- 全新 UI 设计，操作体验流畅
- 对新手更友好
- 持续更新迭代中

### ZL (Zalith Launcher)

- 经典版本，稳定性好
- 资源占用较少
- 适合低配设备

---

选择好启动器后，请在 [下载页面](/docs/download) 获取安装包。
