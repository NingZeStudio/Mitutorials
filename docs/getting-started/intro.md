# 引导/必读

欢迎使用 LogShare & Minecraft: Java Edition for Android 文档！

在开始之前，请仔细阅读以下内容，这将帮助你更好地理解和使用本站提供的信息。

:::tip 💡 核心提示
手机玩 JAVA 版 Minecraft 依赖于**启动器**与**渲染器**的协同工作，上手难度高于网易版。如果你有耐心，手机 JAVA 版圈子欢迎你的加入。
:::

:::warning ⚠️ 性能须知
手机玩 JAVA 版 Minecraft 与其他手游一样，别指望一台性能仅相当于上世纪电脑配置的手机能流畅运行。
:::

## 文档导航

本章节将引导你完成以下步骤：

1. **选择启动器** — 了解 FCL、ZL2、ZL 的区别并选择适合你的启动器
2. **安装与配置** — 下载、安装并完成启动器的基础配置
3. **渲染器设置** — 安装和配置 MG 渲染器等渲染方案
4. **资源管理** — 模组、资源包、光影、整合包的下载与导入
5. **游玩准备** — 启动游戏前的最后检查与设置

## 重要提醒

- **模组兼容性**：并非所有模组都能在手机端运行，部分模组有平台限制或特殊环境要求
- **内存分配**：建议关闭自动分配，原版通常 1G 足够，记住"内存越大，游戏越不稳定"
- **散热**：可以使用散热器或限制帧率上限来避免过热
