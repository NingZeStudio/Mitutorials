# 如何配置MG渲染器（Mobile Glues）？

Mobile Glues（MG）是目前最主流的手机端 Minecraft Java Edition 渲染器，支持模组运行和较好的兼容性。

## 基础配置

1. 确保已安装 MG 渲染器插件
2. 在启动器中将渲染器切换为 MG
3. 根据设备性能调整 MG 参数
