# 如何导入你所喜欢的键位布局？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

如果你已经有下载好的键位布局文件，可以手动导入到启动器中。

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 将布局文件放到 FCL 的布局目录
2. 打开 FCL → **控制布局**
3. 点击**导入**，选择布局文件
4. 应用布局

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 将布局文件放入 ZL2 布局目录
2. 在控制设置中导入
3. 应用布局

</TabItem>
<TabItem value="zl" label="ZL">

1. 复制布局文件到 ZL 布局目录
2. 在设置中加载布局
3. 启用导入的布局

</TabItem>
</Tabs>
