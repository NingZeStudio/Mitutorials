# 如何导入不同的JAVA版本环境？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

不同版本的 Minecraft 对 JAVA 版本有不同要求：

| Minecraft 版本 | 推荐 JAVA 版本 |
|----------------|---------------|
| 1.16.5 及以下 | JAVA 8 |
| 1.17 - 1.20.4 | JAVA 17 |
| 1.20.5+ | JAVA 21 |

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 从 [下载页面](/docs/download) 获取对应版本的 JAVA 运行时
2. 打开 FCL → **设置** → **JAVA 管理**
3. 点击**导入**，选择下载的 JAVA 压缩包
4. 导入完成后，在游戏设置中选择对应的 JAVA 版本

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 下载对应 JAVA 版本
2. 在 ZL2 设置中找到 JAVA 管理入口
3. 导入并选择使用的版本

</TabItem>
<TabItem value="zl" label="ZL">

1. 下载 JAVA 运行时
2. 在设置中导入 JAVA
3. 为游戏版本指定 JAVA

</TabItem>
</Tabs>
