# 如何登录你的微软账号 / 皮肤站账号 / 本地账号？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

FCL 支持以下登录方式：

- **正版登录（微软账号）**：点击右上角账号按钮 → 正版登录 → 粘贴设备码 → 完成认证
- **离线登录**：点击离线登录，输入 3-16 字符 ID
- **第三方登录（皮肤站）**：添加认证服务器地址（如 LittleSkin、NaituSkin）

详细步骤请参考 [FCL 快速入门 - 账号添加](/docs/fcl/intro#账号添加)

:::warning 正版登录须知
微软账号登录需要你已购买并激活 Minecraft 档案。
:::

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 在 ZL2 账号管理中添加账号
2. 选择登录方式（正版/离线/第三方）
3. 输入凭据完成登录

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL 账号设置
2. 选择登录方式
3. 完成账号验证

</TabItem>
</Tabs>
