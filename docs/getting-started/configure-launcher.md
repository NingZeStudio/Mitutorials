# 如何配置你心仪的启动器？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

安装完成后，需要进行基础配置才能正常游玩。

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL，进入**设置**
2. 配置以下关键项：
   - **JAVA 版本**：选择已安装的 JAVA 运行时
   - **内存分配**：建议关闭自动分配，原版 1G 左右
   - **版本隔离**：建议开启
   - **渲染器**：根据需求选择（原版用 kw，模组用 MG）

详细配置请参考 [FCL 快速入门](/docs/fcl/intro#游戏配置)

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 打开 ZL2，进入设置页面
2. 配置 JAVA 路径和内存分配
3. 选择渲染器方案
4. 开启版本隔离

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL，进入设置
2. 设置 JAVA 路径
3. 调整内存分配
4. 选择合适的渲染器

</TabItem>
</Tabs>

:::warning 内存提醒
"内存越大，游戏越不稳定！" — 适度分配即可。
:::
