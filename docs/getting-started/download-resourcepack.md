# 如何下载对应版本的资源包文件？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL → **下载** → **资源包**
2. 选择游戏版本筛选
3. 浏览并下载你喜欢的资源包

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 在 ZL2 下载页面找到资源包分类
2. 筛选版本后下载

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL 资源包下载页面
2. 搜索并下载

</TabItem>
</Tabs>
