# 如何安装你心仪的启动器？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 前往 [FCL 下载站](https://foldcraftlauncher.cn/down.html) 或 [GitHub Releases](https://github.com/FCL-Team/FoldCraftLauncher/releases) 下载最新版 APK
2. 在手机上安装 APK（可能需要允许"未知来源"安装）
3. 打开 FCL，授予必要的权限

:::tip 推荐渠道
[柠枺镜像](https://miawa.cn) 同时提供 FCL、ZL、ZL2 以及渲染器和 JAVA 环境的下载。
:::

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 前往 [ZL 官网](https://www.zalithlauncher.cn/download) 下载 ZL2 安装包
2. 也可以在 [GitHub](https://github.com/ZalithLauncher/ZalithLauncher2/releases) 下载
3. 安装并打开，授予必要权限

</TabItem>
<TabItem value="zl" label="ZL">

1. 前往 [ZL 官网](https://www.zalithlauncher.cn/download) 下载 ZL 安装包
2. 也可以在 [GitHub](https://github.com/ZalithLauncher/ZalithLauncher/releases) 下载
3. 安装并打开，授予必要权限

</TabItem>
</Tabs>
