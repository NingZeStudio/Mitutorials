# 如何导入你所下载的整合包？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL 主界面
2. 进入**版本管理** → **导入整合包**
3. 选择下载的整合包文件（.zip / .mrpack）
4. 等待解析和安装完成
5. 在版本列表中找到新导入的整合包版本

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 在 ZL2 中选择导入整合包
2. 选择文件并等待安装
3. 启动整合包版本

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL 版本管理
2. 导入整合包文件
3. 完成安装后选择并启动

</TabItem>
</Tabs>
