# 如何在你心仪的启动器中下载整合包？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL 主界面
2. 进入**下载** → **整合包**
3. 浏览或搜索你想游玩的整合包
4. 点击下载并等待安装完成

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 在 ZL2 中进入下载中心
2. 找到整合包分类
3. 下载并安装

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL 下载页面
2. 搜索整合包
3. 下载安装

</TabItem>
</Tabs>

你也可以从 [外部渠道](/docs/external-download) 下载整合包后手动导入。
