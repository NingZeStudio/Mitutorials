# 如何导入对应版本的光影文件？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL → **游戏管理** → 选择对应版本
2. 进入**光影管理**
3. 点击**导入**，选择下载的光影文件
4. 在游戏中启用光影

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 将光影文件放入对应版本的 shaderpacks 目录
2. 在游戏内光影设置中启用

</TabItem>
<TabItem value="zl" label="ZL">

1. 复制光影文件到版本目录
2. 在游戏内加载光影

</TabItem>
</Tabs>
