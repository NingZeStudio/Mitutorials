# 如何安装你想游玩的 Minecraft 游戏版本？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL → 左侧边栏 **下载**
2. 选择目标游戏版本
3. 如需模组加载器，选择 Forge/NeoForge/Fabric/Quilt/Optifine
4. 点击下载，等待完成

:::warning 注意
Forge 与 NeoForge 冲突，Fabric 与 Quilt 冲突，不可同时安装。
:::

详细步骤请参考 [FCL 快速入门 - 游戏安装](/docs/fcl/intro#游戏安装)

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 进入 ZL2 版本管理
2. 选择版本和加载器
3. 下载安装

</TabItem>
<TabItem value="zl" label="ZL">

1. 在 ZL 中选择版本下载
2. 按需选择模组加载器
3. 等待安装完成

</TabItem>
</Tabs>
