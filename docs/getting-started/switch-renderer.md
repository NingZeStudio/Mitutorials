# 如何在你心仪的启动器中更换渲染器？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL 主界面
2. 进入 **游戏管理** → **游戏设置**
3. 找到**渲染器**选项
4. 从下拉菜单中选择目标渲染器（kw / MG / Zink）
5. 保存设置

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 打开 ZL2 设置
2. 找到渲染器/图形设置项
3. 选择目标渲染器
4. 应用更改

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL 设置页面
2. 切换渲染器选项
3. 保存生效

</TabItem>
</Tabs>

:::tip 提示
选择渲染器后，建议启动一次游戏验证渲染器是否正常工作。
:::
