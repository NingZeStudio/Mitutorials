# 如何下载对应版本的光影文件？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL → **下载** → **光影**
2. 选择游戏版本筛选
3. 浏览并下载光影包

:::tip 渲染器兼容性
部分光影需要特定渲染器支持，请参考 [光影兼容矩阵](/docs/mobileglues/ShaderSupportMatrix)
:::

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 在 ZL2 中找到光影下载页面
2. 筛选版本后下载

</TabItem>
<TabItem value="zl" label="ZL">

1. 打开 ZL 光影管理
2. 搜索并下载光影包

</TabItem>
</Tabs>
