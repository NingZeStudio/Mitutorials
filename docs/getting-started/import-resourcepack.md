# 如何导入对应版本的资源包文件？

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
<TabItem value="fcl" label="FCL" default>

1. 打开 FCL → **游戏管理** → 选择对应版本
2. 进入**资源包管理**
3. 点击**导入**，选择下载的资源包文件（.zip）
4. 在游戏中启用该资源包

</TabItem>
<TabItem value="zl2" label="ZL2">

1. 将资源包文件放入对应版本的 resourcepacks 目录
2. 在游戏内资源包页面启用

</TabItem>
<TabItem value="zl" label="ZL">

1. 复制资源包到版本目录
2. 在游戏内加载

</TabItem>
</Tabs>
